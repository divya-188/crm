## Color Scheme

### Status Colors
- **Active**: Success green (#10b981)
- **Inactive**: Neutral gray (#6b7280)

### Billing Cycle Colors
- **Monthly**: Primary blue (#3b82f6)
- **Quarterly**: Warning yellow (#f59e0b)
- **Annual**: Success green (#10b981)

### Icon Gradients
- **Primary**: from-primary-500 to-primary-700
- **Success**: from-success-500 to-success-700
- **Neutral**: from-neutral-500 to-neutral-700

## Animations

### Entry Animations
1. **Stats Cards**: Fade in with stagger (100ms delay)
2. **Plan Cards (Grid)**: Scale up with stagger (50ms delay)
3. **Plan Cards (List)**: Slide from left with stagger (30ms delay)
4. **Inline Form**: Fade and slide down from top

### Hover Animations
1. **Plan Cards (Grid)**: translateY(-4px) + shadow-xl
2. **Plan Cards (List)**: shadow-lg
3. **Buttons**: Slight scale and shadow increase

### Interaction Animations
1. **Dropdown**: Scale and fade (95% to 100%)
2. **View Toggle**: Background slide
3. **Form Submit**: Button spinner rotation

## Responsive Breakpoints

### Mobile (< 768px)
- 1 column grid
- Stacked filters
- Hidden stats in list view
- Full-width cards

### Tablet (768px - 1024px)
- 2 column grid
- Some stats hidden
- Horizontal filters

### Desktop (> 1024px)
- 3 column grid
- All features visible
- Optimal spacing
